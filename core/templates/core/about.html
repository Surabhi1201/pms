Signup and Create Account:

Users can sign up for a new account by providing necessary information like name, email address, and password. Upon successful signup, the user's account is created, and they are logged in automatically.

Login:

Users who already have an account can log in using their email address and password. After successful login, users gain access to their dashboard and project management features.

Dashboard:

The dashboard provides an overview of all the projects the user is involved in. Users can see project summaries, no. of tasks on the dashboard.

Projects:

Projects can be edited, updated, and deleted as needed. Each project has its own page where users can view and manage tasks related to that project.

Tasks:

Within each project, users can create tasks with titles, descriptions. Tasks can be edited, updated, and deleted individually. Users can mark tasks as done within the project.
